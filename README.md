# Visual Studio 2022 OpenGL Graphics Template
 A template for starting OpenGL graphics with all dependencies included.

 ---

 ## Dependencies:
 - GLEW 2.1.0
 - GLFW 3.4
 - imgui
 - stb_image
 - glm 0.9.9.8
